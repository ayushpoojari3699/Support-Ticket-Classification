Support Ticket Classification & Prioritization
 Overview

This project builds a machine learning system that automatically classifies support tickets and assigns priority levels.

 Features
Text preprocessing using NLP techniques
TF-IDF vectorization
Ticket classification using Logistic Regression
Priority assignment (High / Medium / Low)
Model evaluation using accuracy and confusion matrix

 Results
Logistic Regression Accuracy: ~85%
Naive Bayes Accuracy: ~76%

 Example
 
Input:
system not working and showing error

Output:
Category: HR Support
Priority: High


 Business Impact
Reduces manual ticket sorting
Improves response time
Helps prioritize critical issues
